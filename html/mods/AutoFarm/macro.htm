Macro

Você pode definir uma determinada ação customizada para ser executada assim que o Auto Farm for encerrado pelo timer.
Caso o character esteja atacando no momento final, o encerramento será adiado até que o monstro seja morto.
%msg%
Teleportar (escape/unstuck)
Desconectar (fechar o jogo)
Usar item %item_name%
Usar skill %skill_name%
